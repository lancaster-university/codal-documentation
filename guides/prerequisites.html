Installation - Tools#

Before we can get started programming the Micro:bit v2 in CODAL, we need to install some tools. The details of how to do this on each operating system differ, so make sure you’re following the right section for your system below.

In summary, we need to have the following components installed:

  • Python 3 or newer

  • Git, via Github Desktop or directly via git-scm

  • Arm GNU Toolchain Downloads

  • CMake

We may also want to install these optional dependencies, which may be useful for specific tasks

  • Visual Studio Code (Recommended!)

  • pyocd or openocd to use on-chip debugging

Linux Instructions#

You should use your distribution’s package repositories to install the required tools. These differ from distribution to distribution, and only a few common ones are listed here, but it is expected that any modern general-purpose distribution should have the required packages to build CODAL.

Debian/Ubuntu (and other APT-based systems)#

In a terminal, update your package index with apt update, then install the basic required packages: apt install build-essential git cmake python3 gcc-arm-none-eabi

Note

Note that these may require root access, via either sudo or entering a root terminal. See your distribution’s package installation documentation for specific details.
