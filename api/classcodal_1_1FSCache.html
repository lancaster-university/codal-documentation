class FSCache#

Public Functions

FSCache(NVMController &nvm, int blockSize, int size = 4)#
Parameters

  • nvm – non - volatile memory controller to use as backing store

  • blockSize – - the size of a logical block, in bytes (n.b. this may be smaller than the physical page size)

  • size – the maximum number of pages in the cache

Retrieves a given block from the cache, if it is present.

Parameters

address – the logical address of the block.

Returns

a pointer to relevent cache entry, or NULL if the block is not cached.

Page a given block into the cache, replacing the LRU block if necessary.

Parameters

address – the logical address of the block to cache.
