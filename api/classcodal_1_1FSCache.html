Write the given area of memory into the buffer provided, paging the data in from FLASH as needed. n.b. data doe NOT need to be word aligned.

Write the given area of memory into the buffer provided, paging the data in from FLASH as needed. Also performs a write-through cache operation directly back if possible.

Parameters

  • address – The logical address of the non-volatile storage to write to. DOES NOT need to be word aligned.

  • data – the data to write.

  • len – amount of data to write, in bytes.

Returns

DEVICE_OK on success, or DEVICE_NOT_SUPPORTED if the attempted operation is not possible without an ERASE operation.
