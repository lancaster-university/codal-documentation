class PacketBuffer#

Class definition for a PacketBuffer. A PacketBuffer holds a series of bytes that can be sent or received from the MicroBitRadio channel.

Note

This is a mutable, managed type.

Public Functions

uint8_t *getBytes()#

Provide a pointer to a memory location containing the packet data.

Returns

The contents of this packet, as an array of bytes.

PacketBuffer()#

Default Constructor. Creates an empty Packet Buffer.

PacketBuffer p();

PacketBuffer(int length)#

Constructor. Creates a new PacketBuffer of the given size.

PacketBuffer p(16);         // Creates a PacketBuffer 16 bytes long.

Parameters

length – The length of the buffer to create.

PacketBuffer(uint8_t *data, int length, int rssi = 0)#

Constructor. Creates an empty Packet Buffer of the given size, and fills it with the data provided.

uint8_t buf = {13,5,2};
PacketBuffer p(buf, 3);         // Creates a PacketBuffer 3 bytes long.

Parameters

  • data – The data with which to fill the buffer.

  • length – The length of the buffer to create.

  • rssi – The radio signal strength at the time this packet was recieved. Defaults to 0.

PacketBuffer(const PacketBuffer &buffer)#

Copy Constructor. Add ourselves as a reference to an existing PacketBuffer.

PacketBuffer p();
PacketBuffer p2(p); // Refers to the same packet as p.

Parameters

buffer – The PacketBuffer to reference.

void init(uint8_t *data, int length, int rssi)#

Internal constructor-initialiser.

Parameters

  • data – The data with which to fill the buffer.

  • length – The length of the buffer to create.

  • rssi – The radio signal strength at the time this packet was recieved.

~PacketBuffer()#

Destructor.

Removes buffer resources held by the instance.

PacketBuffer &operator=(const PacketBuffer &p)#

Copy assign operation.

Called when one PacketBuffer is assigned the value of another using the ‘=’ operator.

Decrements our reference count and free up the buffer as necessary.

Then, update our buffer to refer to that of the supplied PacketBuffer, and increase its reference count.

uint8_t buf = {13,5,2};
PacketBuffer p1(16);
PacketBuffer p2(buf, 3);

p1 = p2;

Parameters

p – The PacketBuffer to reference.

uint8_t operator[](int i) const#

Array access operation (read).

Called when a PacketBuffer is dereferenced with a [] operation.

Transparently map this through to the underlying payload for elegance of programming.

PacketBuffer p1(16);
uint8_t data = p1[0];

uint8_t &operator[](int i)#

Array access operation (modify).

Called when a PacketBuffer is dereferenced with a [] operation.

Transparently map this through to the underlying payload for elegance of programming.

PacketBuffer p1(16);
p1[0] = 42;

bool operator==(const PacketBuffer &p)#

Equality operation.

Called when one PacketBuffer is tested to be equal to another using the ‘==’ operator.

MicroBitDisplay display;
uint8_t buf = {13,5,2};
PacketBuffer p1();
PacketBuffer p2();

if(p1 == p2)                    // will be true
    display.scroll("same!");

Parameters

p – The PacketBuffer to test ourselves against.

Returns

true if this PacketBuffer is identical to the one supplied, false otherwise.

int setByte(int position, uint8_t value)#

Sets the byte at the given index to value provided.

PacketBuffer p1(16);
p1.setByte(0,255);              // Sets the first byte in the buffer to the value 255.

Parameters

  • position – The index of the byte to change.

  • value – The new value of the byte (0-255).

Returns

DEVICE_OK, or DEVICE_INVALID_PARAMETER.

int getByte(int position)#

Determines the value of the given byte in the packet.

PacketBuffer p1(16);
p1.setByte(0,255);              // Sets the first byte in the buffer to the value 255.
p1.getByte(0);                  // Returns 255.

Parameters

position – The index of the byte to read.

Returns

The value of the byte at the given position, or DEVICE_INVALID_PARAMETER.

int length()#

Gets number of bytes in this buffer

PacketBuffer p1(16);
p1.length(); // Returns 16.

Returns

The size of the buffer in bytes.

int getRSSI()#

Retrieves the received signal strength of this packet.

PacketBuffer p1(16);
p1.getRSSI();                 // Returns the received signal strength.

Returns

The signal strength of the radio when this packet was received, in -dbm. The higher the value, the stronger the signal. Typical values are in the range -42 to -128.

void setRSSI(uint8_t rssi)#

Sets the received signal strength of this packet.

PacketBuffer p1(16);
p1.setRSSI(37);
