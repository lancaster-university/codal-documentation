Defines if this filter should perform a deep copy of incoming data, or update data in place.

Parameters

deepCopy – Set to true to copy incoming data into a freshly allocated buffer, or false to change data in place.

virtual void applyEffect(ManagedBuffer inputBuffer, ManagedBuffer outputBuffer, int format)#

Default effect - a simple pass through filter. Override this method in subclasses to create specialist effects/filters.

Parameters

  • inputBuffer – the buffer containing data to process.

  • outputBuffer – the buffer in which to store the filtered data. n.b. MAY be the same memory as the input buffer.

  • format – the format of the data (word size and signed/unsigned representation)
