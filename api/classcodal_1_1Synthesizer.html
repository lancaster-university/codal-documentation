int setFrequency(float frequency)#

Define the central frequency of this synthesizer. Takes effect at the start of the next waveform period.

Parameters

frequency – The frequency, in Hz to generate.

int setFrequency(float frequency, int period, int envelopeStart = 1024, int envelopeEnd = 1024)#

Define the central frequency of this synthesizer. Takes effect at the start of the next waveform period.

Parameters

  • frequency – The frequency, in Hz to generate.

  • period – The period, in ms, to play the frequency.

  • envelopeStart – Starting amplitude (multiplied by general volume of this synth)

  • envelopeEnd – Final amplitude (multiplied by general volume of this synth)

int setVolume(int volume)#

Define the volume of the wave to generate.

Parameters

volume – The new output volume, in the range 0..1023

Returns

DEVICE_OK on success, or DEVICE_INVALID_PARAMETER

int setBufferSize(int size)#

Define the size of the audio buffer to hold. The larger the buffer, the lower the CPU overhead, but the longer the delay.

Parameters

size – The new bufer size to use.

Returns

DEVICE_OK on success, or DEVICE_INVALID_PARAMETER
