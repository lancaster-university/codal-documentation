class MAG3110 : public codal::Compass#

Class definition for a MAG3110 Compass.

Represents an implementation of the Freescale MAG3110 I2C Magnetmometer.

Public Functions

MAG3110(I2C &_i2c, Pin &int1, Accelerometer &_accelerometer, CoordinateSpace &coordinateSpace, uint16_t address = 0x1D, uint16_t id = DEVICE_ID_COMPASS)#

Constructor. Create a software representation of an e-compass.

MAG3110 compass(i2c, accelerometer, coordinateSpace);

Parameters

  • _i2c – an instance of codal::I2C, which the compass is accessible from.

  • int1 – The codal::Pin connected to the MAG3110 interrupt line.

  • _accelerometer – an instance of codal::Accelerometer, used for tilt compensation.

  • coordinateSpace – The 3D coordinate space transform to use with this sensor.

  • address – the address of the MAG3110 device on the i2c bus. Defaults to MAG3110_DEFAULT_ADDR.

  • id – the ID of the new MAG3110 object. Defaults to MAG3110_DEFAULT_ADDR.

int whoAmI()#
virtual int configure() override#

Configures the compass for the sample rate defined in this object. The nearest values are chosen to those defined that are supported by the hardware. The instance variables are then updated to reflect reality.

Returns

DEVICE_OK or DEVICE_I2C_ERROR if the magnetometer could not be configured.

virtual int requestUpdate() override#

Updates the local sample, only if the compass indicates that data is stale.

Note

Can be used to trigger manual updates, if the device is running without a scheduler. Also called internally by all get[X,Y,Z]() member functions.

virtual void idleCallback() override#

Periodic callback from the idle thread.

Calls updateSample().
