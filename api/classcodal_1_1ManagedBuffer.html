class ManagedBuffer#

Class definition for a ManagedBuffer. A ManagedBuffer holds a series of bytes for general purpose use. n.b. This is a mutable, managed type.

Public Functions

ManagedBuffer()#

Default Constructor. Creates an empty ManagedBuffer. The ‘ptr’ field in all empty buffers is shared.

Example: 

ManagedBuffer p();

ManagedBuffer(int length, BufferInitialize initialize = BufferInitialize::Zero)#

Constructor. Creates a new ManagedBuffer of the given size.

Example: 

ManagedBuffer p(16);         // Creates a ManagedBuffer 16 bytes long.

Parameters

length – The length of the buffer to create.

ManagedBuffer(uint8_t *data, int length)#

Constructor. Creates an empty ManagedBuffer of the given size, and fills it with the data provided.

Example: 

uint8_t buf[] = {13,5,2};
ManagedBuffer p(buf, 3);         // Creates a ManagedBuffer 3 bytes long.

Parameters

  • data – The data with which to fill the buffer.

  • length – The length of the buffer to create.

ManagedBuffer(const ManagedBuffer &buffer)#

Copy Constructor. Add ourselves as a reference to an existing ManagedBuffer.

Example: 

ManagedBuffer p();
ManagedBuffer p2(i);        // Refers to the same buffer as p.

Parameters

buffer – The ManagedBuffer to reference.

ManagedBuffer(BufferData *p)#

Constructor. Create a buffer from a raw BufferData pointer. It will ptr->incr(). This is to be used by specialized runtimes.

Parameters

p – The pointer to use.

void initEmpty()#

Internal constructor helper. Configures this ManagedBuffer to refer to the static empty buffer.

void init(uint8_t *data, int length, BufferInitialize initialize)#

Internal constructor-initialiser.

Parameters

  • data – The data with which to fill the buffer.

  • length – The length of the buffer to create.

  • initialize – The initialization mode to use for the allocted memory in the buffer

~ManagedBuffer()#

Destructor. Removes buffer resources held by the instance.

inline uint8_t *getBytes()#

Provide an array containing the buffer data.

Returns

The contents of this buffer, as an array of bytes.

BufferData *leakData()#

Get current ptr, do not decr() it, and set the current instance to an empty buffer. This is to be used by specialized runtimes which pass BufferData around.

ManagedBuffer &operator=(const ManagedBuffer &p)#

Copy assign operation.

Called when one ManagedBuffer is assigned the value of another using the ‘=’ operator. Decrements our reference count and free up the buffer as necessary. Then, update our buffer to refer to that of the supplied ManagedBuffer, and increase its reference count.

Example: 

uint8_t buf = {13,5,2};
ManagedBuffer p1(16);
ManagedBuffer p2(buf, 3);

p1 = p2;

Parameters

p – The ManagedBuffer to reference.

inline uint8_t operator[](int i) const#

Array access operation (read).

Called when a ManagedBuffer is dereferenced with a [] operation. Transparently map this through to the underlying payload for elegance of programming.

Example: 

ManagedBuffer p1(16);
uint8_t data = p1[0];

inline uint8_t &operator[](int i)#

Array access operation (modify).

Called when a ManagedBuffer is dereferenced with a [] operation. Transparently map this through to the underlying payload for elegance of programming.

Example: 

ManagedBuffer p1(16);
p1[0] = 42;

bool operator==(const ManagedBuffer &p)#

Equality operation.

Called when one ManagedBuffer is tested to be equal to another using the ‘==’ operator.

Example: 

uint8_t buf = {13,5,2};
ManagedBuffer p1(16);
ManagedBuffer p2(buf, 3);

if(p1 == p2)                    // will be true
    uBit.display.scroll("same!");

Parameters

p – The ManagedBuffer to test ourselves against.

Returns

true if this ManagedBuffer is identical to the one supplied, false otherwise.

int setByte(int position, uint8_t value)#

Sets the byte at the given index to value provided.

Example:

ManagedBuffer p1(16);
p1.setByte(0,255);              // Sets the first byte in the buffer to the value 255.

Parameters

  • position – The index of the byte to change.

  • value – The new value of the byte (0-255).

Returns

DEVICE_OK, or DEVICE_INVALID_PARAMETER.

int getByte(int position)#

Determines the value of the given byte in the buffer.

Example: 

ManagedBuffer p1(16);
p1.setByte(0,255);              // Sets the first byte in the buffer to the value 255.
p1.getByte(0);                  // Returns 255.

Parameters

position – The index of the byte to read.

Returns

The value of the byte at the given position, or DEVICE_INVALID_PARAMETER.

inline int length() const#

Gets number of bytes in this buffer

Example:

ManagedBuffer p1(16);
p1.length();                 // Returns 16.

Returns

The size of the buffer in bytes.
