class Timer#

Public Functions

Timer(LowLevelTimer &t, uint8_t ccPeriodChannel = 0, uint8_t ccEventChannel = 1)#

Constructor for a generic system clock interface.

Parameters

  • t – A low-level timer instance that wraps a hardware timer.

  • ccPeriodChannel – The channel to use a a periodic interrupt as a fallback

  • ccEventChannel – The channel to use as the next TimerEvents’ interrupt.

int eventAfter(CODAL_TIMESTAMP period, uint16_t id, uint16_t value, uint32_t flags = 0)#

Configures this Timer instance to fire an event after period milliseconds.

Parameters

  • period – the period to wait until an event is triggered, in milliseconds.

  • id – the ID to be used in event generation.

  • value – the value to place into the Events’ value field.

  • flags – CODAL_TIMER_EVENT_FLAGS_WAKEUP for event to trigger deep sleep wake-up.

int cancel(uint16_t id, uint16_t value)#

Cancels any events matching the given id and value.

  • value – the value that was given upon a previous call to eventEvery / eventAfter
