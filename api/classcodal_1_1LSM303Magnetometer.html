  • id – The unique EventModel id of this component. Defaults to: DEVICE_ID_ACCELEROMETER

virtual int configure() override#

Configures the magnetometer for the sample rate defined in this object. The nearest values are chosen to those defined that are supported by the hardware. The instance variables are then updated to reflect reality.

Note

This method should be overidden by the hardware driver to implement the requested changes in hardware.

Returns

DEVICE_OK on success, DEVICE_I2C_ERROR if the magnetometer could not be configured.

virtual int requestUpdate() override#

Poll to see if new data is available from the hardware. If so, update it. n.b. it is not necessary to explicitly call this function to update data (it normally happens in the background when the scheduler is idle), but a check is performed if the user explicitly requests up to date data.

Note

This method should be overidden by the hardware driver to implement the requested changes in hardware.

Returns

DEVICE_OK on success, DEVICE_I2C_ERROR if the update fails.

virtual void idleCallback() override#

A periodic callback invoked by the fiber scheduler idle thread.

Internally calls updateSample().

Puts the component in (or out of) sleep (low power) mode.
