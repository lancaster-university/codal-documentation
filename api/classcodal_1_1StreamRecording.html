float duration(unsigned int sampleRate)#

Calculate the recorded duration for this StreamRecording.

As this cannot be known by this class (as the sample rate may change during playback or recording) the expected rate must be supplied.

Parameters

sampleRate – The sample rate to calculate the duration for, in samples per second.

Returns

long The total duration of this StreamRecording, based on the supplied sample rate, in seconds.

bool isFull()#

Checks if this object can store any further ManagedBuffers from the upstream components.

Note

This does not mean that RAM is completely full, but simply that there is now more internal storage for ManagedBuffer references.

Returns

true If there are no more slots available to track more ManagedBuffers.

Returns

false If there is remaining internal storage capacity for more data

bool record()#

Begin recording data from the connected upstream.

The StreamRecording object will, if already playing; stop playback, erase its buffer, and start recording.

Non-blocking, will return immediately.

Returns

Do not use this value, return semantics are changing.

bool play()#

Begin playing data from the connected upstream.

The StreamRecording object will, if already recording; stop recording, rewind to the start of its buffer, and start playing.

Non-blocking, will return immediately.

Returns

Do not use this value, return semantics are changing.

void stop()#

Stop recording or playing the data stored in this StreamRecording object.

Repeated calls to this will do nothing if the object is not in a recording or playback state.

Returns

Do not use this value, return semantics are changing.

void erase()#

Erase the internal buffer.

Will also stop playback or recording, if either are active.

Returns

Do not use this value, return semantics are changing.

bool isPlaying()#

Checks if the object is playing back recorded data.

Returns

True if playing back, else false if stopped or recording.

bool isRecording()#

Checks if the object is recording new data.

Returns

True if recording, else false if stopped or playing back.

bool isStopped()#

Checks if the object is stopped.

Returns

True if stopped, else false if recording or playing back.
