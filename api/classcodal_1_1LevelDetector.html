Public Functions

LevelDetector(DataSource &source, int highThreshold, int lowThreshold, uint16_t id = DEVICE_ID_SYSTEM_LEVEL_DETECTOR, bool connectImmediately = true)#

Creates a component capable of measuring and thresholding stream data

Parameters

  • source – a DataSource to measure the level of.

  • highThreshold – the HIGH threshold at which a LEVEL_THRESHOLD_HIGH event will be generated

  • lowThreshold – the HIGH threshold at which a LEVEL_THRESHOLD_LOW event will be generated

  • id – The id to use for the message bus when transmitting events.

  • connectImmediately – Should this component connect to upstream splitter when started

int setWindowSize(int size)#

Set the window size to the given value. The window size defines the number of samples used to determine a sound level. The higher the value, the more accurate the result will be. The lower the value, the more responsive the result will be. Adjust this value to suit the requirements of your applicaiton.

Parameters

size – The size of the window to use (number of samples).

Returns

DEVICE_OK on success, DEVICE_INVALID_PARAMETER if the request fails.
