class DataStream : public codal::DataSource, public codal::DataSink#

Class definition for DataStream. A Datastream holds a number of ManagedBuffer references, provides basic flow control through a push/pull mechanism and byte level access to the datastream, even if it spans different buffers.

void setBlocking(bool isBlocking)#

Determines if this stream acts in a synchronous, blocking mode or asynchronous mode. In blocking mode, writes to a full buffer will result int he calling fiber being blocked until space is available. Downstream DataSinks will also attempt to process data immediately as it becomes available. In non-blocking asynchronpus mode, writes to a full buffer are dropped and downstream Datasinks will be processed in a new fiber.
