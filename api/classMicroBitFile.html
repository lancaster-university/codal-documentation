Class MicroBitFile#

Class Documentation#

class MicroBitFile#

Public Functions

MicroBitFile(ManagedString fileName, int mode = 0x01 | 0x02 | 0x04)#

Creates an instance of a MicroBitFile and creates a new file if required.

Parameters

  • fileName – the name of the file to create/open.

  • mode – One of: READ, WRITE, READ_AND_WRITE. Defaults to READ_AND_WRITE.

int setPosition(int position)#

Seeks to a position in this MicroBitFile instance from the beginning of the file.

Parameters

offset – the number of bytes to seek, relative to the beginning of the file.

Returns

the new seek position, MICROBIT_NOT_SUPPORTED if the current file handle is invalid, MICROBIT_INVALID_PARAMETER if the given offset is negative.

int getPosition()#

Returns the current position of the seek head for the current file.

Returns

the new seek position, MICROBIT_NOT_SUPPORTED if the current file handle is invalid.

int write(const char *bytes, int len)#

Writes the given bytes to this MicroBitFile instance at the current position.

Parameters

  • bytes – a pointer to the bytes to write to this file.

  • len – the number of bytes to write.

Returns

The number of bytes written, MICROBIT_NOT_SUPPORTED if the current file handle is invalid, MICROBIT_INVALID_PARAMETER if bytes is invalid, or len is negative.

int write(ManagedString s)#

Writes the given ManagedString to this MicroBitFile instance at the current position.

Parameters

s – The ManagedString to write to this file.

Returns

The number of bytes written, MICROBIT_NOT_SUPPORTED if the current file handle is invalid, MICROBIT_INVALID_PARAMETER if bytes is invalid, or len is negative.

int read()#

Reads a single character from the file at the current position.

Returns

the character, or MICROBIT_NOT_SUPPORTED if the current file handle is invalid.

int read(char *buffer, int size)#

Reads from the file into a given buffer.

Parameters

  • buffer – a pointer to the buffer where data will be stored.

  • size – the number of bytes that can be safely stored in the buffer.

Returns

the number of bytes read, or MICROBIT_INVALID_PARAMETER if buffer is invalid, or the size given is less than 0.

ManagedString read(int size)#

Reads from the current MicroBitFile into a given buffer.

Parameters

size – the number of bytes to be read from the file.

Returns

a ManagedString containing the requested bytes, oran empty ManagedString on error.

int remove()#

Removes this MicroBitFile from the MicroBitFileSystem.

Returns

MICROBIT_OK on success, MICROBIT_INVALID_PARAMETER if the given filename does not exist, MICROBIT_CANCELLED if something went wrong.

int append(const char *bytes, int len)#

Seeks to the end of the file, and appends the given ManagedString to this MicroBitFile instance.

Parameters

  • bytes – a pointer to the bytes to write to this file.

  • len – the number of bytes to write.

Returns

The number of bytes written, MICROBIT_NOT_SUPPORTED if the current file handle is invalid or this file was not opened in WRITE mode.

int append(ManagedString s)#

Seeks to the end of the file, and appends the given ManagedString to this MicroBitFile instance.

Parameters

s – The ManagedString to write to this file.

Returns

The number of bytes written, MICROBIT_NOT_SUPPORTED if the current file handle is invalid or this file was not opened in WRITE mode.

bool isValid()#

Determines if this MicroBitFile instance refers to a valid, open file.

Returns

true if this file is valid, false otherwise.

int getHandle()#

Returns the handle used by this MicroBitFile instance.

Note

This member function will also inform the user of any errors encountered during the opening of this MicroBitFile. At open, the handle is set to the return value from MicroBitFileSystem.open().

int close()#

Closes this MicroBitFile instance

Note

MicroBitFiles are opened at construction and are implicitly closed at destruction. They can be closed explicitly using this member function.

Returns

MICROBIT_OK on success, MICROBIT_NOT_SUPPORTED if the file handle is invalid.

int flush()#

Writes back all state associated with the given file to FLASH memory, leaving the file open.

Returns

MICROBIT_OK on success, MICROBIT_NOT_SUPPORTED if the file system has not been initialised or if this file is invalid.

~MicroBitFile()#

Destructor for MicroBitFile. Implicitly closes the current file.
