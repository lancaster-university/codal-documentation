void play(const void *data, int length, int count = 1)#

Perform a blocking playout of the data buffer. Returns when all the data has been queued.

Parameters

  • data – pointer to memory location to playout

  • length – number of samples in the buffer. Assumes a sample size as defined by setFormat().

  • count – if set, playback the buffer the given number of times. Defaults to 1. Set to a negative number to loop forever.
