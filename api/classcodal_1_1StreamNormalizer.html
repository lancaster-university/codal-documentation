class StreamNormalizer : public codal::DataSink, public codal::DataSource#

Public Functions

StreamNormalizer(DataSource &source, float gain = 1.0f, bool normalize = false, int format = 0, int stabilisation = 0)#

Creates a component capable of translating one data representation format into another

Parameters

  • source – a DataSource to receive data from

  • gain – The gain to apply to each sample (default: 1.0)

  • normalize – Derive a zero offset for the input stream, and subtract from each sample (default: false)

  • format – The format to convert the input stream into

  • stabilisation – the maximum change of zero-offset permitted between subsequent buffers before output is initiated. Set to zero to disable (default)
