int flash_write(void *address, void *buffer, int length, void *scratch_addr = NULL)#

Writes the given number of bytes to the address in flash specified. Neither address nor buffer need be word-aligned.

Example:

MicroBitFlash flash();
uint32_t word = 0x01;
flash.flash_write((uint8_t*)0x38000, &word, sizeof(word))

Parameters

  • address – location in flash to write to.

  • buffer – location in memory to write from. @length number of bytes to burn

  • scratch_addr – if specified, scratch page to use. Use default otherwise.

Returns

non-zero on sucess, zero on error.

void flash_burn(uint32_t *page_address, uint32_t *buffer, int len)#

Write to flash memory, assuming that a write is valid (using need_erase).

Parameters

  • page_address – address of memory to write to. Must be word aligned.

  • buffer – address to write from, must be word-aligned.

  • len – number of uint32_t words to write.
